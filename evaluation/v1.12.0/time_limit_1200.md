# Evaluation of MIPLIB 2017 (time_limit = 1200)

## miplib-12

1. The results are listed in the [miplib-12/README.md](miplib-12/README.md) file.

2. The content of `highs_option.txt` file is the following:
```ini
time_limit = 1200
parallel = off
solver = simplex
simplex_strategy = 1
mip_rel_gap = 0
mip_abs_gap = 1e-06
```
